# 评委演示路径

## 网页体验

1. 打开仓库主页部署的在线演示入口。
2. 进入 `NearbyLove 三端协同体验`，选择居民、物业或居委会身份。
3. 居民端查看通知、附近行动与社区广场；通过 AI 社区管家了解服务或被引导写入社区信箱。
4. 物业端查看设施运行地图、履约任务、AI 场景信与通知反馈纪要。
5. 居委会端查看区域关怀地图、行动协调与 AI 工作摘要。
6. 返回入口，进入 `CommunitySense 政府治理平台` 查看区域总览与分社区信息。

## 可选后端演示

1. 启动 Flask API。
2. 运行 `scripts/seed_demo_areas.py` 写入模拟公共空间读数。
3. 访问 `/api/v1/areas/overview` 或网页大屏观察区域级汇总。

所有演示读数均为模拟数据。
